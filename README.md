# คลังยา โรงพยาบาลบางจาก ปี 2569

เว็บค้นหาและ Dashboard ข้อมูลยาที่รวบรวมจากเอกสาร PDF จำนวน 12 ชุดในโฟลเดอร์แม่ รองรับการค้นหาจากชื่อยา ความแรง รูปแบบยา ยี่ห้อ และกลุ่มเอกสาร พร้อมรายละเอียดมาตรการจำกัดการใช้ เกณฑ์ DUE และแหล่งอ้างอิงกลับไปยังเอกสารต้นฉบับ

## สรุปข้อมูล

- ข้อมูลต้นทาง 904 แถว
- รายการหลังรวมข้อมูลซ้ำ 708 รายการ
- กลุ่มค้นหา 12 กลุ่ม
- ข้อมูลหน้าเว็บอยู่ที่ `public/data/drugs.json`

## เปิดดูในเครื่อง

```bash
npm install
npm run dev
```

## อัปเดตข้อมูลจาก PDF

วางไฟล์ PDF ทั้ง 12 ไฟล์ไว้ในโฟลเดอร์แม่ของ `drug-dashboard` แล้วเรียกสคริปต์ต่อไปนี้ในสภาพแวดล้อมที่ติดตั้ง `pdfplumber`:

```bash
python3 scripts/extract_drug_data.py
```

สคริปต์จะอ่านตาราง รวมรายการซ้ำ และสร้าง `public/data/drugs.json` ใหม่โดยอัตโนมัติ

## เผยแพร่ GitHub Pages

Workflow ใน `.github/workflows/deploy-pages.yml` จะสร้างและเผยแพร่หน้าเว็บทุกครั้งที่ push ไปยัง branch `main` หลังสร้าง repository ให้ตั้งค่า **Settings → Pages → Source** เป็น **GitHub Actions**

> ข้อมูลนี้ใช้เพื่อการสืบค้น กรุณาตรวจสอบเอกสารต้นฉบับและดุลยพินิจของแพทย์หรือเภสัชกรก่อนใช้ยา
